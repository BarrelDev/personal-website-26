## Overview

YouTube Scraper is a utility that interfaces with the YouTube Data API to fetch and aggregate the latest videos from a user's subscription feed. Built with JavaScript, it provides a streamlined way to keep up with subscribed channels without relying on YouTube's built-in recommendation algorithm. The tool authenticates via the YouTube Data API, retrieves subscription lists, and pulls recent uploads from each channel into a consolidated feed.
