## Overview

Tilted is an endless platformer game built with the Unity engine. Players navigate a character through procedurally generated platforms, aiming to travel as far as possible without falling. The game features progressively increasing difficulty, responsive controls, and a scoring system that tracks the player's best runs. Developed in C#, the project explores Unity's 2D physics, procedural generation techniques, and game state management.
