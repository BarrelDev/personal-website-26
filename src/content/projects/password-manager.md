## Overview

Password Manager is a lightweight, local-first utility for securely storing and managing passwords. Built entirely in Python, it prioritizes simplicity and privacy by keeping all data on the user's machine rather than relying on cloud services. The application provides basic CRUD operations for credential entries and uses encryption to protect stored passwords at rest.
